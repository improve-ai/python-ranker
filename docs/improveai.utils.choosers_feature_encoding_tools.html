choosers_feature_encoding_tools.py module

encoded_variant_to_np(encoded_variant, feature_names)

Puts values from encoded_variant into numpy array. Only those features in encoded_variant which overlap with feature_names. Resulting array must contain all values stored in feature_names. Features missing in encoded_variants are filled with np.nan.

Parameters

  • encoded_variant (dict) – fully encoded single variant

  • feature_names (list or np.ndarray) – array of feature names from model which will be used for predictions

Returns

single row of a shape (1, num. features) which contains fully encoded variant

Return type

np.ndarray

encoded_variants_to_np(encoded_variants, feature_names)

Converts an array of variants into 2D numpy array (<N variants>, <M features>). Only those features in a single encoded variants which overlap with feature_names will be present in the resulting matrix. Resulting array must contain all values stored in feature_names. Features missing in encoded_variants are filled with np.nan.

Parameters

  • encoded_variants (np.ndarray) – array of encoded variants

  • feature_names (np.ndarray) – array of feature names from model which will be used for predictions

Returns

array of (num. variants, num. features) shape

Return type

np.ndarray
