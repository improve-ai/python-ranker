_encode(obj, path, into, noise_shift=0.0, noise_scale=1.0)ÔÉÅ

Encodes a JSON serializable object to a float vector Rules of encoding go as follows:

  • None, json null, {}, [], and nan are treated as missing features and ignored.

  • numbers and booleans are encoded as-is.

  • strings are encoded using a lookup table

Parameters:

  • obj (object) ‚Äì a JSON serializable object to be encoded to a flat key-value structure

  • path (str) ‚Äì the JSON-normalized path to the current object

  • into (np.ndarray) ‚Äì an array into which feature values will be encoded

  • noise_shift (float) ‚Äì small bias added to the feature value

  • noise_scale (float) ‚Äì small multiplier of the feature value
